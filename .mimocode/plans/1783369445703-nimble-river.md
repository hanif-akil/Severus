# Clipboard History Panel Implementation Plan

## Overview

Add a clipboard history panel to the Simple Keyboard app. The panel shows up to 20 recent clipboard items, allows tapping to paste, and includes a delete button per item. A new clipboard key in the keyboard's bottom row opens the panel.

## Architecture

The feature follows the existing MoreKeysPanel pattern: a panel view is inflated from XML, shown above the keyboard via the `DrawingPreviewPlacerView`, and dismissed on tap-outside or item selection.

## New Files

### 1. `app/src/main/java/rkr/simplekeyboard/inputmethod/latin/ClipboardHistory.java`
Singleton that monitors `ClipboardManager.OnPrimaryClipChangedListener` and stores up to 20 text `ClipData` items in memory (as `String` list). Methods: `getItems()`, `addItem(String)`, `removeItem(int)`, `clear()`. Initialized in `LatinIME.onCreate()`.

### 2. `app/src/main/java/rkr/simplekeyboard/inputmethod/keyboard/ClipboardHistoryView.java`
A custom `ViewGroup` (extends `LinearLayout` or `RecyclerView`) that displays clipboard items. Each item is a row with truncated text and a delete (X) button. Tapping an item pastes it via `KeyboardActionListener.onTextInput()`. Implements a simple `showInParent()`/`removeFromParent()` pattern similar to `MoreKeysPanel`. The panel is shown as a popup above the keyboard.

### 3. `app/src/main/res/layout/clipboard_history_panel.xml`
Layout for the clipboard history panel. A `LinearLayout` with a title bar ("Clipboard") and a scrollable list area. Each item row is inflated dynamically.

### 4. `app/src/main/res/layout/clipboard_history_item.xml`
Layout for a single clipboard item row: a `TextView` (truncated text) + an `ImageButton` (delete icon).

### 5. `app/src/main/res/drawable/sym_keyboard_clipboard.xml`
Vector drawable for the clipboard key icon (Material Design clipboard icon, 24dp).

## Modified Files

### 6. `Constants.java` (line ~106)
Add: `public static final int CODE_CLIPBOARD_HISTORY = -14;`

### 7. `KeyboardCodesSet.java` (line ~57)
Add `"key_clipboard_history"` to `ID_TO_NAME[]` and `Constants.CODE_CLIPBOARD_HISTORY` to `DEFAULT[]`.

### 8. `KeyboardIconsSet.java` (line ~81)
Add `NAME_CLIPBOARD_HISTORY_KEY = "clipboard_history_key"` and map it to `R.drawable.sym_keyboard_clipboard`.

### 9. `key_styles_common.xml` (line ~111)
Add new key style:
```xml
<key-style
    latin:styleName="clipboardHistoryKeyStyle"
    latin:keySpec="!icon/clipboard_history_key|!code/key_clipboard_history"
    latin:keyActionFlags="noKeyPreview"
    latin:backgroundType="functional" />
```

### 10. `row_qwerty4.xml`
Replace or add the clipboard key. The bottom row currently has: symbol, comma, space, period, enter. Add a clipboard key before the symbol key, shifting widths:
- Clipboard key (8%p) | Symbol (12%p) | Comma | Space | Period | Enter

### 11. `InputLogic.java` (line ~178-221)
Add `case Constants.CODE_CLIPBOARD_HISTORY:` in `handleFunctionalEvent()` that calls `mLatinIME.toggleClipboardHistoryPanel()`.

### 12. `KeyboardActionListener.java`
Add method: `void onToggleClipboardHistory();` and implement in `Adapter`.

### 13. `LatinIME.java`
Add `toggleClipboardHistoryPanel()` method that shows/hides the clipboard panel. Initialize `ClipboardHistory` in `onCreate()`. Implement `onToggleClipboardHistory()` callback.

### 14. `MainKeyboardView.java`
Add `showClipboardHistoryPanel()` and `dismissClipboardHistoryPanel()` methods. The panel is shown by inflating `clipboard_history_panel.xml`, populating items from `ClipboardHistory`, and adding it to the `DrawingPreviewPlacerView`.

## Implementation Order

1. Add `CODE_CLIPBOARD_HISTORY` to Constants, KeyboardCodesSet, KeyboardIconsSet
2. Create clipboard icon drawable
3. Add `clipboardHistoryKeyStyle` to key_styles_common.xml
4. Create `ClipboardHistory.java` singleton
5. Create `clipboard_history_panel.xml` and `clipboard_history_item.xml` layouts
6. Create `ClipboardHistoryView.java`
7. Add clipboard key to `row_qwerty4.xml` and other bottom rows
8. Wire up `InputLogic`, `LatinIME`, `MainKeyboardView`, and `KeyboardActionListener`
9. Test the feature end-to-end

## Verification

1. Build the project with `./gradlew assembleDebug`
2. Install on device/emulator and test:
   - Clipboard key appears in bottom row
   - Tapping clipboard key opens the panel
   - Panel shows "Clipboard" title and empty state when no items
   - Copy text in another app, return to keyboard, open panel - item appears
   - Tap item to paste into text field
   - Delete item via X button
   - Panel dismisses when tapping outside
   - Panel works in both portrait and landscape
   - Panel works with different keyboard themes
